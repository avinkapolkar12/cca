About the Project

OpenStack is an open-source software platform for cloud computing, mostly deployed as infrastructure-as-a-service (IaaS). It enables you to create and manage large groups of virtual private servers.

Key Features

  • Scalable and Modular Architecture
  • Supports Public, Private, and Hybrid Clouds
  • Open-source and Vendor Neutral
